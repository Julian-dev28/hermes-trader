# W-C4 candle_pattern_family — short-side reversal effect is REAL; engulf not a lone fluke

## Hypothesis
If only engulf survives, C9 is overfit; if the 2-bar reversal family survives, it's real.

## Rule
Daily cross-sectional, hz=1, stop 0.40. Patterns: engulf, harami, piercing/dark-cloud,
hammer/shooting-star. For each: SYM + LONG-leg + SHORT-leg, scoring each leg as EXCESS over
a matched same-side random null (long-pool / short-pool), OOS both halves.

## Read
- **Every LONG/bullish reversal leg is dead** (excess ≤ +0.20%, mostly negative, OOS-broken).
  The reversal effect is one-sided: bearish-reversal → short only. Reinforces W-C2.
- **The SHORT side is a partial but real family:** engulf-short is strongest and OOS-robust
  (p=0.0002, both halves +); **harami-short independently corroborates** (p=0.031, both
  halves +, just weaker h2). Two distinct 2-bar patterns that both flag a current bar
  rejecting a prior up-move clear the same-side null → engulf is NOT a lone overfit fluke.
- The family is not uniform: piercing/dark-cloud has high raw excess but **sign-flips OOS**
  (h1 +3.46 / h2 −0.92 → not robust), and hammer/shooting-star is dead. So it's a specific
  "strong bearish body engulfing/containing a prior up-body" effect, not generic candlestick
  magic.

## VERDICT
**Family PARTIALLY survives — short-side only — which LOWERS the overfit flag.** Deciding
number: engulf-short excess **+1.25% (p=0.0002, OOS-robust)** is corroborated by an
independent cousin, harami-short **+0.61% (p=0.031, both halves +)**; all long legs and
hammer/star are dead, piercing fails OOS. C9 is one (the cleanest) instance of a real
bearish-reversal-→-short effect, not a single overfit pattern. Confirms the SHORT-ONLY
wire from W-C2.
